# git_edit
Simple text editor for Git.

## Features
- Quick launch.
- Syntax highlighting:
    - `COMMIT_EDITMSG` with diff (See ``git commit -v``).
    - `.gitignore`
- Input completion:
    - ``Ctrl+Space`` to open the window.
- ``Ctrl+Q`` to save and exit.
